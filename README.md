# 📚 StudyFlow — Student Assignment Planner

A clean, minimal assignment planner built with **pure HTML, CSS, and JavaScript** — zero dependencies, zero build tools, one file.

---

## ✨ Features

- ➕ Add, edit, and delete assignments
- ✅ Mark assignments complete / undo
- 🔴 Overdue detection with visual alerts
- 🔍 Filter by priority, status, and search
- 📊 Analytics bar (total, completed, pending, overdue, high priority)
- 💾 Data saved in `localStorage` — persists across page reloads
- 📱 Fully responsive (mobile-friendly)
- ⚡ Zero frameworks, zero npm, zero build step

---

## 🗂️ Project Structure

```
studyflow/
```

## 💡 Concepts Used

| Concept | Where |
|---|---|
| localStorage | Persist assignments across sessions |
| DOM manipulation | Rendering all UI with vanilla JS |
| Event delegation | Button clicks on cards |
| Array methods | filter, find, map for data operations |
| CSS variables | Consistent design tokens |
| CSS Grid | Responsive card layout |
| CSS animations | fadeUp, slideIn on page/card load |
| Hash routing | Simple page navigation without a library |
